Windows：怎么获取 “当前播放歌曲”
方法 A：系统媒体控制接口（最稳，支持所有播放器）
Windows 有 GlobalSystemMediaTransportControls (GSMTC)，所有现代播放器（网易云、Spotify、QQ 音乐、Apple Music）都会把播放信息同步到这里。
Rust 实现（用 winrt 库）
toml
# Cargo.toml
[
dependencies
]
winrt = "0.7"
winrt-runtime = "0.7"
tokio = { version = "1", features = ["full"] }
rust
运行
// 导入WinRT媒体控制命名空间
winrt::include_bindings!();
use windows::media::control::*;
use windows::core::*;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    // 1. 获取系统媒体会话管理器
    let manager = GlobalSystemMediaTransportControlsSessionManager::request_async()?.await?;
    
    loop {
        // 2. 获取当前播放会话
        if let Some(session) = manager.get_current_session() {
            // 3. 获取媒体信息（歌名、歌手）
            if let Ok(props) = session.try_get_media_properties_async()?.await {
                let title = props.title()?.to_string();
                let artist = props.artist()?.to_string();
                println!("🎵 正在播放: {title} - {artist}");
            }
            // 4. 获取播放进度（秒）
            if let Ok(timeline) = session.get_timeline_properties() {
                let pos = timeline.position()?.total_seconds();
                println!("⏱️ 进度: {pos:.1}s");
            }
        }
        tokio::time::sleep(tokio::time::Duration::from_millis(500)).await;
    }
}
